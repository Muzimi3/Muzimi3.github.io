  2. 开始下载:
    • 点击”Download for free”

  3. 等待下载完成:
    • 注意这里下载的不是软件，是安装包，安装包不大，因此下载很快

在安装的时候

这几个选项最好选上，后续使用它打开文件夹更方便。安装完后，我们找到一个python项目，右键，选择使用Cursor打开

  • 首次启动时，需要登录或创建一个 Cursor 账户

点击sign up，输入你的名字和邮箱。关于邮箱这个，进入到这个网址生成临时邮箱：https://temp-mail-docs.awsl.uk/
使用临时邮箱进行注册，Cursor对于新用户会有14天的Pro使用(内含150次快速响应机会)，当你14天过期或150次用完，继续重新注册新临时邮箱进行白嫖

后面设置完密码后，你的临时邮箱会收到验证码（刷新你的临时邮箱界面），填入即可。注册完成后进入下面页面（页面不要关掉）

返回Cursor点击左上角的文件->首选项->Cursor setting

然后进入到Cursor账户页面，点击sign in

它会跳转到浏览器（你刚刚注册Cursor账户的浏览器)，点击Log in

后面会出现：

返回Cursor

安装对应的扩展（重要）

点击这里安装扩展：

安装后可以汉化 +选择已经有的python虚拟环境+运行python文件（完全代替Pycharm）

在安装完python扩展后需要选择虚拟环境：先选择显示状态栏

当我们点击一个python文件后，状态栏那里会出现pythonXXX，我们点击它，会弹出你已经创建好的虚拟环境，即可选择。

我们可以使用快捷键Ctrl+L呼出Chat窗口，可以选择对应的模型，首推Claude3.7-sonnet(gpt都是垃圾)

在聊天框中输入对应问题：

  • 你也可以在输入框中输入 @ 然后选择你想要引用的内容，或者手动添加希望关联的文件。
  • 你也可以直接复制一个url链接，甚至直接复制一个图片，cursor支持图片识别。
  • 当我们输入@的时候，会弹出你问题需要关联的单个文件或文件夹或整个项目，可以@file1 @file2 @codebase 然后输入问题等等

Curosr有预测功能，它会根据你写的代码，预测你下一步要做什么，如果它预测正确，我们可以直接按Tab键接收即可，例如我像给下面函数写注释，我刚打了##，它会预测我会写注释，并且智能生成注释，我只需要按Tab接收即可

Apply代码块

  1. 当AI生成代码建议时,您会看到代码块右上角的”Apply”按钮。
  2. 点击”Apply”将代码直接应用到您的文件中。
  3. 应用后,您可以:
    • 使用 Ctrl/⌘ + Enter 接受更改。
    • 使用 Ctrl/⌘ + Backspace 拒绝更改。

文件中会显示修改内容，绿色的表示这次修改的内容，红色的表示原来的内容，当点击accept时，表示接受这次的修改
